function

キャラクターがアシスタント

キャラクターがアドバイスやメッセージをくれます。また、達成度によってキャラクターの表情が変化します。

カレンダーでチートデイを確認

アプリを開くとカレンダーが表示され、事前に登録しておいたチートデイを一目で確認することができます。

買うものリストの作成

レシピを検索しながら、必要な材料を買うものリストに追加することができます。右下の鉛筆のアイコンから入力することも可能です。

料理の記録

自分が作った料理を写真に撮ってアップロードするだけで、簡単に記録をすることができます。使用したレシピも一緒に記録できるので、気にいった料理は再び検索をしなくてもすぐに見つけることが可能です。
